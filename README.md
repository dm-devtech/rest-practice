# REST Practice

Let's practice REST in a Rails context

## Quickstart

```bash
git clone <this repo>
cd rest-practice
bundle
rails s
```

## Quickstart (Docker)

```bash
git clone <this repo>
cd rest-practice
docker build -t rest-practice .
docker run -p 3000:3000 -v $(pwd):/usr/src/app rest-practice
```

Then visit http://localhost:3000 and you should see an amazing homepage
